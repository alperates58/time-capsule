# Monetization

Bu doküman TimeCapsule Project Bible v2 içinde yer alır. Amaç, Antigravity/Gemini/Codex gibi AI araçlarının projeyi tutarlı şekilde geliştirmesini sağlamaktır.

## Gelir Modelleri
- Display ads
- Affiliate links
- Premium no-ads
- API access
- Educational plans
- Sponsored timeline modules
- Historical reports

## Reklam İlkesi
Reklamlar deneyimi bozmamalı. Sayfa değerini düşüren reklam yoğunluğu yasak.
