# Security

Bu doküman TimeCapsule Project Bible v2 içinde yer alır. Amaç, Antigravity/Gemini/Codex gibi AI araçlarının projeyi tutarlı şekilde geliştirmesini sağlamaktır.

## Başlangıç Güvenliği
- Admin panel protected
- Environment variables güvenli
- Rate limiting
- Input validation
- CSRF/XSS önlemleri
- DB migration review

## Veri Güvenliği
Public data ağırlıklı olsa da admin hesapları ve import pipeline korunmalıdır.
