# Search Strategy

Bu doküman TimeCapsule Project Bible v2 içinde yer alır. Amaç, Antigravity/Gemini/Codex gibi AI araçlarının projeyi tutarlı şekilde geliştirmesini sağlamaktır.

## Arama Kapsamı
Kullanıcı yıl, kişi, film, oyun, olay, şirket veya kategori arayabilir.

## İlk Aşama
PostgreSQL full-text search + trigram.

## Ölçek Sonrası
Meilisearch veya Typesense.

## Search UX
- Instant search
- Year suggestions
- Entity suggestions
- Category filters
- Fuzzy matching
- Synonyms / aliases
