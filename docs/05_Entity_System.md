# Entity System

Bu doküman TimeCapsule Project Bible v2 içinde yer alır. Amaç, Antigravity/Gemini/Codex gibi AI araçlarının projeyi tutarlı şekilde geliştirmesini sağlamaktır.

## Entity Nedir?
TimeCapsule'daki her anlamlı şey entity'dir.

## Entity Tipleri
Movie, Series, Game, Album, Song, Book, Person, Company, Country, Event, War, Award, Phone, Car, Software, Website, AI Model, Space Mission, Sport Event.

## Entity Alanları
- id
- type
- title
- slug
- short_description
- long_description
- start_date
- end_date
- country_id
- canonical_image
- source_confidence
- publication_status

## Entity Sayfası
Her entity kendi sayfasına sahip olabilir. Örneğin `/movie/titanic-1997`, `/person/steve-jobs`, `/game/half-life`.
